**# social-media-analytics**# Instagram Social Media Analytics & ETL

This repository contains a Python-based pipeline for processing and analyzing Instagram data. The project facilitates the consolidation of raw scraped data from multiple accounts into a unified dataset, followed by bivariate statistical analysis to identify correlations between various engagement metrics.

## 📌 Project Overview

The project is structured into three main phases:
1.  **ETL (Extract, Transform, Load):** Consolidating disparate CSV files (`user.csv` and `posts.csv`) from individual user directories into a single master dataset.
2.  **Feature Engineering:** Extracting specific attributes such as emojis from post captions for deeper content analysis.
3.  **Bivariate Analysis:** Using statistical modeling and visualizations (regression plots) to understand the relationship between account reach (followers/followings) and user engagement (likes/comments).

## 📂 Project Structure

* **`etl.ipynb`**: The entry point for data processing. It walks through the `/insta` directory, merges user metadata with post-specific data, and cleans the final output.
* **`analisis_bivariado.ipynb`**: Contains the statistical analysis. It uses Seaborn and Scipy to calculate linear regressions, $R^2$ values, and slopes for different variables.
* **`translate.ipynb`**: a utility notebook for translating captions and text using the Google Translate API (Experimental).
* **`dataset.csv`**: The master file generated by the ETL process, used as the source for all analyses.

## 🚀 Usage

### 1. Data Preparation
Place your raw Instagram scrapings in a folder named `/insta`. Each subfolder should represent a user and contain:
* `user.csv`: Contains `username`, `followers`, `followings`, `posts_count`, etc.
* `posts.csv`: Contains `caption`, `likes`, `comments`, `img_url`, etc.

### 2. Running ETL
Run the `etl.ipynb` notebook. This will:
* Walk through all user folders.
* Join user metadata to every individual post.
* Extract emojis into a separate column.
* Save the result as `dataset.csv`.

### 3. Statistical Analysis
Open `analisis_bivariado.ipynb` to visualize the data. The notebook generates regression plots and prints key metrics:
* **$R^2$ Score:** To determine how much of the engagement variance is explained by the follower count.
* **Slope:** To understand the growth rate of likes/comments per new follower.

## 📊 Sample Analysis Insights

The current analysis focuses on:
* **Likes vs. Followers:** Identifying the "Engagement Rate" through linear regression.
* **Comments vs. Followers:** Determining if higher reach leads to higher conversation rates.
* **Followings vs. Likes:** Observing if the "Follow-for-Follow" strategy impacts post-performance.
